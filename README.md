# KBO 스탯 센터

KBO 리그 팀 순위, 타자/투수 기록, 선수 정보를 한눈에 보는 웹사이트입니다.

- **데이터 출처**: [KBO 공식 홈페이지](https://www.koreabaseball.com) (팀 순위, 선수 기록, 연도별 기록)
- **이미지**: 네이버 스포츠 CDN 고화질(선수 210×262, 팀 로고 184×184) 우선, KBO 공식 이미지로 자동 폴백
- **구성**: Python 수집 스크립트 + React(Vite) 웹앱

## 폴더 구조

```
KBO_statics/
├── scraper/
│   └── scrape_kbo.py     # KBO 데이터 수집 스크립트
└── web/                  # React 웹앱
    ├── public/data/      # 수집된 JSON 데이터 (스크립트가 생성)
    └── src/              # 앱 소스코드
```

## 데이터 갱신하기

**현재 시즌** 최신 기록으로 업데이트 (10~15분):

```
py scraper/scrape_kbo.py
```

**역대 시즌**(1982~직전 시즌) 수집 — 최초 1회만 필요, 이미 수집된 연도는 건너뜀:

```
py scraper/scrape_history.py
```

완료 후 웹페이지를 새로고침하면 반영됩니다.

## 웹사이트 실행하기

```
cd web
npm run dev
```

브라우저에서 http://localhost:5173 접속.

배포용 정적 파일을 만들려면 `npm run build` (결과물: `web/dist/`).

## 페이지 안내

| 페이지 | 내용 |
|---|---|
| 팀 순위 | 순위표, 승률 바, 최근 10경기, 연승/연패 — **1982년부터 시즌 선택 가능** |
| 타자 기록 | 타율·홈런·타점·OPS 등 8개 부문 리더보드 + 전체 선수 정렬/검색 — 시즌 선택 가능 |
| 투수 기록 | 평균자책점·승리·탈삼진·세이브·홀드 등 8개 부문 리더보드 — 시즌 선택 가능 |
| 선수 상세 | 사진, 프로필(생년월일·포지션·경력), 시즌 성적, 연도별 기록 + 통산 |
| 구단 페이지 | 팀별 전체 로스터 (투수/포수/내야수/외야수), 순위표에서 팀 클릭 |
| 선수 비교 | 타자끼리/투수끼리 2명 비교, 우위 항목 하이라이트 |
| 용어 가이드 | 야구 기록 용어를 초보자 눈높이로 설명 (예시 포함) |

비율 기록(타율, 평균자책점 등)의 순위는 규정타석/규정이닝을 채운 선수만 포함됩니다.
